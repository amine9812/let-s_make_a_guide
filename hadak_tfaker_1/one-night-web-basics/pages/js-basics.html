JavaScript makes things react.

We use JS to read values, make decisions, and update the DOM.

Quick reference

  • let or const to store data.
  • querySelector grabs elements.
  • addEventListener reacts to clicks and typing.

COMMON MISTAKE: Forgetting document.querySelector returns one element. Use querySelectorAll for many.

Variables, arrays, and objects

const name = "Ada";

let score = 0; (use let when it will change)

const colors = ["red", "blue"];

const person = { name: "Lee", age: 20 };

Loops and conditions

for (const color of colors) { ... } runs through each color.

if (score > 10) { ... } else { ... } makes choices.
