HTML builds the skeleton of a page.

HTML is just boxes and labels. CSS paints them. JS moves them.

Mini recap

  • Doctype tells the browser this is HTML5.
  • Head holds meta info (like the title).
  • Body holds what people see.

Document structure

Key tags you should know:

  • <h1> to <h6>: headings, like chapter titles.
  • <p>: paragraphs for regular text.
  • <a>: links. Needs an href to go somewhere.
  • <img>: images. Needs src and alt.
  • <ul> / <ol> with <li>: lists.

Semantic tags

These tags describe meaning, not style. They help screen readers and search engines.

<header>, <nav>, <main>, <section>, <article>, <footer>.

Form basics

Forms collect user input. Labels connect text to inputs.

Why use labels? They make forms accessible and enlarge the clickable area.

Why required? Browser can stop empty submissions for free.
