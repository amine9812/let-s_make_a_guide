Step 2 • Style

CSS paints the boxes.

Selectors find elements. Properties change how they look and sit on the page.

Core ideas

  • Use classes most of the time.
  • IDs are unique. Avoid styling many IDs.
  • The box model: content → padding → border → margin.

COMMON MISTAKE: Forgetting that padding and borders add to element size.

Selectors

  • Element: p { color: blue; }
  • Class: .note { color: green; }
  • ID: #title { font-size: 2rem; }

TRY THIS: Add the class note to a paragraph in the HTML file and reload.

Pseudo-classes

These apply when something special happens.

  • a:hover when the mouse is over a link.
  • input:focus when an input is active.
  • button:disabled when a button cannot be used.
