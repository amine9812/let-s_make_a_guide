How this works

  1. Open index.html in your browser.
  2. Follow the pages in the top nav.
  3. Edit code snippets and sliders to see instant results.
  4. Track your progress with the checkboxes.

TIP: Keep the browser and a text editor open side by side.

1. HTML basics

Learn the skeleton of a page: headings, paragraphs, links, lists, images, and semantic layout tags.

TRY THIS: Change a heading and watch the preview update.

2. CSS basics

Style the page with colors, spacing, borders, Flexbox, and hover/focus states.

TRY THIS: Move the range sliders to change padding and roundness.

3. JavaScript basics

Add behavior: variables, functions, events, and DOM updates. Save small bits of state with localStorage.

TRY THIS: Click the counter buttons and read the code comments.

4. Playground

Mix everything together. Build a profile card with friendly controls for HTML, CSS, and JS tweaks.

TRY THIS: Toggle dark mode and see how classes change the style.
